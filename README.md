# Cheftrack Website

Production-ready Next.js codebase for the Cheftrack marketing website and legal pages.

## Tech Stack

- Next.js
- React
- TypeScript
- Vercel-ready app router setup

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import the repository into Vercel.
3. Use the default Next.js settings.
4. Deploy.

## Structure

- `app/` Next.js routes
- `components/` reusable React components
- `content/landing-page.html` migrated landing markup
- `lib/` content loaders and legal content
- `public/` static assets
